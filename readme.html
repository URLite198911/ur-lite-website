UR Lite README

What is UR Lite?

UR Lite is a lightweight local Windows document reader. It opens documents on the user's own computer and is designed for simple reading, fast navigation, and clean viewing of common document and ebook formats.

Supported formats

  • PDF
  • EPUB
  • DOCX
  • TXT
  • CBR
  • CBZ

Installing UR Lite

Download the UR Lite installer from the purchase or download page when available. Run the installer on Windows and follow the prompts.

Opening files

Open UR Lite, choose a supported file, and begin reading. Large files may take longer to load, especially very large PDFs, DOCX files, or comic archives.

Reading PDF files

UR Lite opens PDF files for local reading with simple navigation and viewing controls.

Reading EPUB files

UR Lite supports EPUB reading and includes theme options such as light, sepia, and dark.

Reading DOCX files

DOCX files can open in HTML/Reflow View without LibreOffice. This view is intended for readable text flow and may not preserve every detail of the original page layout.

DOCX Original Layout requirement

UR Lite can use LibreOffice for DOCX Original Layout viewing. LibreOffice must be installed locally on the computer. If LibreOffice is not available, UR Lite opens DOCX files in HTML/Reflow View.

Download LibreOffice from the official website

Reading CBR and CBZ comic archives

CBZ files are ZIP-based comic archives. 7-Zip may help with archive handling.

CBR files are RAR-based comic archives. WinRAR or compatible RAR support may be needed.

Helper software links

For safety, download helper software only from the official websites below.

  • LibreOffice
  • 7-Zip
  • WinRAR

Troubleshooting

DOCX does not show original formatting
Install LibreOffice and reopen the DOCX file.
CBR file does not open
Install WinRAR or compatible RAR support.
CBZ file does not open
Install 7-Zip.
Large files open slowly
Very large PDFs, DOCX files, or comic archives may take longer to load.
Documents are not uploaded
UR Lite opens files locally on the computer.

Known limitations

Some document layouts and archive formats depend on the file itself and the helper software available on the computer. DOCX Original Layout requires LibreOffice. CBR files may require WinRAR or compatible RAR support.

Support

For help, contact support@universalreaderlite.com. Include the UR Lite version, Windows version, file type, and any visible error message.
