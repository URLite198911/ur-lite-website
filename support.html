Support

For UR Lite support, contact support@universalreaderlite.com.

Before contacting support

Please include the details below when possible. They help identify the issue more quickly.

  • UR Lite version
  • Windows version
  • File type being opened
  • Screenshot of the error if available
  • Whether LibreOffice, 7-Zip, or WinRAR is installed when relevant

Helper software

Use the official websites for helper software downloads:

  • LibreOffice
  • 7-Zip
  • WinRAR
